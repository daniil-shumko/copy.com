How to talk to the API

To upload images using API, simple make a POST request with a "multipart/form-data" form.
The form should have:
"caption" - with a type="text" and max number of characters should be 128.
"category" - should be one of the category ID's(not the category name!) See above table for all categories and their ID's.
"image" - of a type="file" and should not exceed 10mb in size.(Supported image formats are: JPG, JPEG, PNG and GIF)

The POST request should be sent to "http://daniilshumko.pythonanywhere.com/image_hosting/api/"

Responses from the API

If image has been uploaded successfully you will receive a JSON response with the information about the image you have uploaded.

{
"success" : "true",
"image_url" : "http://daniilshumko.pythonanywhere.com/media/images/image_name.jpg",
"caption" : "Your caption text",
"category" : "Category name"
}

 If there is an error you will receive the following response.
{
"success" : "false",
"error" : "Error message."
}
